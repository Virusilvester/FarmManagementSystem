## Overview

The Farm Management System is a comprehensive C# application demonstrating 
object-oriented programming principles for managing farm operations.

## Features

- ✅ Animal management (Cows, Chickens, Sheep)
- ✅ Crop cultivation (Wheat, Corn, Vegetables)
- ✅ Production and sales tracking
- ✅ Action history and audit trails
- ✅ Day simulation

## Quick Start

1. Clone the repository
2. Open in Visual Studio or your preferred C# IDE
3. Run Program.cs
4. Follow the interactive menu
